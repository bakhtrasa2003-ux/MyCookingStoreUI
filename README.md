A modern Restaurant Food Ordering App UI built using Java and XML layouts in Android Studio.
The app includes a splash screen, category-based food listing using RecyclerView, and screen navigation using Intents.

Features;
Splash Screen with smooth transition
Modern and clean UI design
Food items displayed using RecyclerView
Intent-based navigation between activities
Food details screen

XML-based responsive layouts;
Add to cart UI (basic implementation)

Tech Stack;
Language: Java
UI Design: XML Layouts
Architecture: Activity-based structure
Navigation: Intents
UI Components: RecyclerView, CardView, ConstraintLayout

App Flow;
Splash Screen Activity
Displays app logo
Automatically navigates to Home Activity using Intent
Home Activity
Displays food items in RecyclerView
Each item shows image, name, price, and rating
